HTML Audio/Video DOM

Utilizando HTML y JavaScript, se pide practicar con las etiquetas <video> y <audio> vistas en clase para control de video/audio. El objetivo es familiarizarse con el manejo de propiedades, método y eventos vistos en clase.

  • EJERCICIO:Crear botones para iniciar, y pausar un vídeo o audio
  • EJERCICIO: Añadir controles al vídeo por HTML. Utillizar javascript para activar/desactivar los controles del vídeo.Mostrar por pantalla/alert/console el estado de los controles:true/false.
  • EJERCICIO: Mostrar por pantalla/alert/console un mensaje cuando el usuario pulse play/pause en el vídeo
  • EJERCICIO: Mostrar por pantalla/alert/console cuando el usuario cambie volumen de vídeo
  • EJERCICIO: En la vista HTML, se pide cambiar de vídeo pulsando un botón. Es necesario tener dos vídeos para que se alternen cada vez que se pulse un botón. load().
  • EJERCIO: Duración del vídeo. Pulsando un botón se debe poder cambiar el tiempo de ejecución del vídeo. Ejemplo: si pulso el botón se visualizará a partir del segundo 20.
  • EJERCICIO PENDIENTE: cambiar el código para no tener que utilizar el atributo onclick en los botones del HTML y hacer la llamada a los métodos desde el fichero JavaScript. Usar de referencia el ejercicio de audio: controls_audio.html
