Ejemplo del uso del sistema de cuadrícula de Bootstrap para repartir dos textos en forma de columnas usando "div" y clase "container-fluid"

Este ejemplo mostrará los "div" en proporción 25%/75% para dispositivos pequeños y 50%/50% para medianos/grandes dispositivos. Para pequeños dispositivos los "div" se redimensionarán al 100%.

Expande/contrae la pantalla para ver el efecto.
